# Assets

This folder is for public-facing visual materials used in GitHub demos, portfolio summaries, and case studies.

## Recommended assets

- workflow diagrams
- GH screenshots
- Rhino screenshots
- building generator previews
- demo GIF
- final portfolio boards

## Suggested structure

- `workflow/`
- `grasshopper/`
- `rhino/`
- `generator_previews/`
- `demo_gif/`
- `portfolio_boards/`

## Do not upload

- raw DWG files
- raw Rhino files
- original assignment briefs from teachers or studios
- personal information
- API keys
- private client or school data
- unprocessed old project archives

## Notes

Assets should support the workflow story. Prefer screenshots that show data-to-model steps, GH wiring, separate Custom Preview outputs, summary Panels, and human validation moments.
