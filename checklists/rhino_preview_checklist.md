# Rhino Preview Checklist

- Site boundary appears in the expected location.
- Roads align with site and parcels.
- Parcel curves and labels match IDs.
- Height points match height rules.
- Massing breps appear where expected.
- Towers, podiums, open spaces, roofs, and facade lines are previewed separately.
- Materials or colors make output categories readable.
- No duplicate preview from Python default Preview.
- No hidden failure behind `all_breps`.
- Summary panel reports expected counts.
- Human designer accepts or rejects the viewport result.
