# GitHub Release Checklist

- README uses `Spatial AI Modeling System` as the title.
- README explains the human-in-the-loop workflow.
- Product manifest is complete.
- System manifest explains layers.
- Skills are present and consistently structured.
- Schemas are present.
- Templates are present.
- Validators run on sample data.
- Examples use fictional data only.
- Prompt library is included.
- Failure cases and lessons are documented.
- Screenshots or demo GIFs are added before public release.
- Claims avoid saying AI automatically designs buildings.
- Portfolio summary and resume bullets are reviewed by the human author.
