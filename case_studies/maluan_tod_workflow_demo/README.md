# Maluan TOD Workflow Demo

This is a sanitized workflow case study for the Spatial AI Modeling System.

It demonstrates how the system can be applied to a real AI-assisted spatial modeling workflow, while keeping raw project assets private. The previous project experience is used only as a workflow reference. This case study does not copy or expose original DWG, Rhino, Grasshopper, raw site data, assignment files, or private project assets.

## What this case study shows

- How site data can be prepared for spatial modeling.
- How JSON rules can structure parcel hierarchy, height rules, and design intent.
- How Codex can support GH Python cell generation, validation scripts, and documentation.
- How Rhino/GH preview is used to verify geometry outputs.
- How ChatGPT can help with building generator logic and visual debugging prompts.
- How human validation closes the loop before any output is considered usable.
